# Hotel Management System Database Project

This project is a database system for managing hotel operations such as rooms, guests, bookings, and employees.

## Project Idea

The system helps manage:

- Room availability
- Guest information
- Booking records
- Employee information
- Occupancy and booking reports

## Main Tables

- Rooms
- Guests
- Bookings
- Employees

## How to Run

Run the SQL DDL files first, then insert or test the data.

Recommended order:

1. Rooms
2. Guests
3. Employees
4. Bookings

Bookings should be created after Rooms and Guests because it depends on them through foreign keys.
